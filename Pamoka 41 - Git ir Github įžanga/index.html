Git

Git - populiariausia kodo versijavimo programa.

Git Komandos be configu

  • init - sukuria repozitoriją
  • add - pridedama nurodytus arba visus failus(stage)
  • status - nurodo lokalios git repozitorijos statusą. "Your branch is up to date with 'origin/master'"" - nurodo, kad tavo kodas naujausias
  • commit -m "žinutė" - sukuria kodo versiją iš addintu(stagintu) failu ir prideda žinutę apie pakitimus.(žinutė iki 50 simbolių)
  • push - iš lokalios repozitorijos nauju pakeitmų išsiuntimas į išorinę repozitoriją
  • pull - iš išorinės atsisiuntimas į lokalią
  • remote add orige- dažniausiai vykdoma vien kartą prijungiant lokalią prie išorinės
  • log - comitų sąrašą, q - išeiti
  • checkout branchName|commitId - išsikvieti pakeitima - cheout master - grįžimas prie pagridinio
  • branch naujaŠaka - sukuria šaką
  • git push --set-upstream origin branch1 - supušinti naują branchą
    Pirmasis Github paleidimas per Git:
  • git init - paleisti naują git repositoriją - nevykdyti jau esamos git repositorijos viduje
  • git status - patikrinti pakitimus
  • git add . - prideti pirmus failus. "."- reikia visus faulus su visais katalogais.
  • git commit -m 'pirmas komitas' - komitinimas su žinute (m)
  • git remote add origin 'https://github.com/dariussh77/Test2' - prijungia pri GIthub
  • git push -u origin master - atsiranda jau Githube

Github

Github- viena iš svetainių kelti versijuotą kodą.

Sukurti nauja repositorija per Github. Pasirenkame vardą, komentarą. Git.ignore - pasirinkti failus, kurių gitas neseka.
